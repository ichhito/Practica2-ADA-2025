# Practica2-ADA-2025
Repositorio para la segunda práctica de la asignatura Análisis y Diseño de Algoritmos


# Objetivo del trabajo
Dado una lista de contenedores, dos contenedores son concatenables si en el contenedor A tiene una reordenacion donde cada atributo de este es estrictamente menor que otro B.

El objetivo de la práctica es encontrar el contenedor de mayor longitud posible, usando algortimo voraz, dinámica y backtracking.
